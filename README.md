# RegreSModelling



## 🏠 House Price Prediction using Regression

This project applies **regression modeling techniques** to predict house prices based on various features such as area, number of bedrooms, and location characteristics. The project demonstrates **data preprocessing**, **simple & multiple linear regression**, **model evaluation**, and **visualization** using Python and `scikit-learn`.

---

### 📊 Dataset Description

The dataset contains the following features:

* `area`: Size of the house (in square feet)
* `bedrooms`: Number of bedrooms
* `bathrooms`: Number of bathrooms
* `stories`: Number of floors
* `mainroad`: Whether the house is on a main road (`yes`/`no`)
* `guestroom`: Whether the house has a guest room (`yes`/`no`)
* `basement`: Whether the house has a basement (`yes`/`no`)
* `hotwaterheating`: Availability of hot water heating
* `airconditioning`: Availability of air conditioning
* `parking`: Number of parking spaces
* `prefarea`: Preferred area (`yes`/`no`)
* `furnishingstatus`: Furnishing status (`furnished`, `semi-furnished`, `unfurnished`)
* `price`: Target variable — house price (in currency units)

---

### 📌 Key Features

* ✅ **Simple Linear Regression** (One feature)
* ✅ **Multiple Linear Regression**
* ✅ **Model evaluation** using:

  * Mean Absolute Error (MAE)
  * Mean Squared Error (MSE)
  * R² Score
* ✅ **Plots**:

  * Actual vs Predicted prices
  * Regression line for single-feature models
* ✅ **Categorical feature encoding**
* ✅ **Train-test split**
